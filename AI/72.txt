You've hit on a very important nuance! It's not an either/or situation, but rather that one enables the other.

You're asking: "Is it not the one-to-one relation with N that makes it countable, it is the nature of nominator and denominator are discrete numbers makes you able to make it countable correct?"

Here's the precise explanation:

1.  **The Definition of Countable:**
    A set is defined as **countably infinite** *if and only if* you can establish a **one-to-one correspondence** (a bijection) between its elements and the set of natural numbers (N = {1, 2, 3,...}). This means you can create an infinite list where every element of the set appears exactly once, and each element in the list is assigned a unique natural number (1st, 2nd, 3rd, etc.).

2.  **How the Nature of Numerators and Denominators Enables Countability:**
    You are absolutely correct that the **discrete nature of integers** (which serve as numerators and denominators for rational numbers) is precisely *what allows us to create* that one-to-one correspondence.

    Because integers are discrete and countable themselves, we can systematically list all possible pairs of integers (a, b) where `b ≠ 0`. Think of the grid method we discussed:
    *   You can list all integers for the numerator (0, 1, -1, 2, -2,...)
    *   You can list all non-zero integers for the denominator (1, -1, 2, -2,...)

    By creating a systematic path through this grid (like the zig-zag method, skipping duplicates), you can ensure that every single rational number `a/b` will eventually be reached and assigned a unique position in an infinite list. This ability to list them is what makes them countable.

**So, to summarize:**

*   **What makes a set countable?** The existence of a one-to-one correspondence with the natural numbers.
*   **How is this possible for rational numbers?** Because their components (integer numerators and denominators) are discrete and countable, allowing for a systematic listing method.

The discrete nature of integers provides the "mechanism" or "scheme" by which the rational numbers can be put into a one-to-one correspondence with the natural numbers, thus fulfilling the definition of countability. This is in stark contrast to real numbers, which include irrational numbers that cannot be systematically listed in this manner.
