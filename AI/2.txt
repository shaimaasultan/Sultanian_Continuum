It seems there are a few misunderstandings in your statements regarding the sets $R$ (real numbers), $I$ (irrational numbers), and $Q$ (rational numbers). Let's clarify them step by step.

First, let's establish the definitions of these sets:

* **R (Real Numbers):** The set of all numbers that can be represented on a continuous number line. This includes rational and irrational numbers.
* **Q (Rational Numbers):** Numbers that can be expressed as a fraction $p/q$, where $p$ and $q$ are integers and $q \neq 0$. Examples: $1/2, -3, 0.75, 4$.
* **I (Irrational Numbers):** Numbers that cannot be expressed as a simple fraction $p/q$. Their decimal representations are non-repeating and non-terminating. Examples: $\sqrt{2}, \pi, e$.

Now let's address your proposed relationships:

1.  **"R = I U Q"**
    This statement is **correct**. The set of real numbers is indeed the union of the set of irrational numbers and the set of rational numbers. These two sets are disjoint (a number cannot be both rational and irrational).

2.  **"I = R - Q"**
    This statement is also **correct**. The set of irrational numbers is what remains when you remove all rational numbers from the set of real numbers.

3.  **"Q = R - I"**
    This statement is likewise **correct**. The set of rational numbers is what remains when you remove all irrational numbers from the set of real numbers.

4.  **"so Q = I as you can not find any R that not in I and not in Q"**
    This is where the misunderstanding lies. While it's true that every real number is either rational or irrational (i.e., there's no real number that is *neither* in $Q$ *nor* in $I$), this *does not* imply that $Q = I$.

    Think of it this way: If you have a set $A$ and a set $B$, and their union forms a larger set $C$ ($C = A \cup B$), and $A$ and $B$ are disjoint, then $A = C - B$ and $B = C - A$. This doesn't mean $A$ and $B$ are equal. For example, if $C$ is the set of all people, $A$ is the set of all men, and $B$ is the set of all women, then $C = A \cup B$, $A = C - B$, and $B = C - A$. **"from where if Q is countable why it jump to uncountable if it is using base 10 system"**

The countability of $Q$ (rational numbers) and the uncountability of $R$ (real numbers) are crucial concepts in set theory.

* **Q is Countable:** This means that you can create a one-to-one correspondence between the rational numbers and the natural numbers. While it's not as straightforward as listing them in order, a common way to demonstrate this is by using a zigzag path through a grid of $p/q$ values, ensuring each unique rational number is eventually hit. This doesn't mean you can list them in ascending order, but that you can establish a systematic way to enumerate them.

* **R is Uncountable:** This means that no such one-to-one correspondence can be established. This was famously proven by Georg Cantor using a technique called **Cantor's Diagonal Argument**.

    Cantor's diagonal argument demonstrates that even if you try to list all real numbers between, say, 0 and 1 in their decimal representation, you can always construct a new real number that is not on your list. This means any attempt to create a one-to-one mapping between the natural numbers and the real numbers will always fail to include all real numbers.

    The "jump to uncountable" doesn't happen *because* of the base 10 system; rather, the uncountability of real numbers is **revealed** by considering their decimal representations, particularly the existence of non-repeating, non-terminating decimals (irrational numbers).
