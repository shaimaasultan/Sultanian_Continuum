That's a very insightful leap you're making, and it touches on a common point of confusion when people are first learning about cardinality!

However, the conclusion "then they both are countable" is **incorrect**.

Here's why:

1.  **A mapping from an uncountable set to a countable set is always possible.**
    * What you've described is a scenario where a specific operation (a kind of function or mapping) takes an input that involves an irrational number (from an uncountable set, like $\mathbb{I}$ or $\mathbb{R}$) and produces an output that is a natural number (from a countable set, $\mathbb{N}$).
    * Consider a very simple example: the **floor function**, denoted $\lfloor x \rfloor$. This function takes any real number $x$ and gives you the greatest integer less than or equal to $x$.
        * $\lfloor \pi \rfloor = 3$
        * $\lfloor \sqrt{2} \rfloor = 1$
        * $\lfloor 5.999 \rfloor = 5$
        * $\lfloor -2.1 \rfloor = -3$
    * This function maps the entire uncountable set of **real numbers ($\mathbb{R}$)** onto the countable set of **integers ($\mathbb{Z}$)**.
    * The existence of such a mapping (from a larger set to a smaller/equal sized set) **does not mean the larger set is countable.** It only means the smaller set is no larger than the larger one.

2.  **Countability requires a *bijection* (one-to-one and onto).**
    * To prove that two sets have the same cardinality (meaning they are both countable, or both uncountable with the same degree of infinity), you need to establish a **bijection** between them. A bijection is a mapping where:
        * **Every element in the first set maps to a *unique* element in the second set (one-to-one).**
        * **Every element in the second set is mapped to by *some* element in the first set (onto).**
    * The specific cases you've found (like $\pi + D(\pi) = 3$ or $e^{i\pi} = -1$) are not bijections from the set of all irrational numbers (or all complex numbers) to the set of natural numbers.
        * Many different irrational numbers might map to the same natural number under a given operation (e.g., both $3.1$ and $3.01$ map to 3 under the floor function). This means it's not one-to-one.
        * Such specific operations also don't necessarily cover *all* natural numbers, and they certainly don't cover all irrational or complex numbers.

3.  **The Uncountability Proof Stands.**
    * The uncountability of the real numbers (and thus the irrational numbers) is rigorously proven by Georg Cantor's Diagonal Argument. This proof demonstrates that no matter how you try to construct a one-to-one mapping between the natural numbers and the real numbers, you will always be able to construct a real number that you missed. This proof remains valid regardless of specific functions or identities.

So, while it's fascinating to see how certain operations bridge different number sets, the existence of such specific mappings does not alter the fundamental cardinality of those sets. The set of real numbers (and irrational numbers) remains uncountably infinite.
