If you could find a one-to-one mapping (a bijection) between the set of **Irrational Numbers (I)** and the set of **Rational Numbers (Q)**, it would indeed imply that the set of **Real Numbers (R)** is **countable**.

Here's the breakdown of why:

1.  **Definition of Countability:** A set is considered **countably infinite** if its elements can be put into a one-to-one correspondence with the natural numbers (N = {1, 2, 3,...}). The cardinality of such sets is denoted as ℵ₀ (aleph naught).

2.  **Cardinality of Rational Numbers (Q):** The set of rational numbers (Q) is known to be countably infinite. This means |Q| = ℵ₀.

3.  **The Hypothetical:** Your question proposes a scenario where there is a one-to-one mapping between Irrational Numbers (I) and Rational Numbers (Q). If such a mapping existed, it would mean that the cardinality of the irrational numbers is the same as the cardinality of the rational numbers, i.e., |I| = |Q|. Since |Q| = ℵ₀, this would imply |I| = ℵ₀.

4.  **Real Numbers as a Union:** The set of **Real Numbers (R)** is the union of the set of rational numbers and the set of irrational numbers (R = Q ∪ I).

5.  **Cardinality of the Union:** If both Q and I were countably infinite (|Q| = ℵ₀ and |I| = ℵ₀), then the cardinality of their union (R) would also be countably infinite. In set theory, the union of two countably infinite sets is still countably infinite (ℵ₀ + ℵ₀ = ℵ₀).

**Conclusion:**

Therefore, **yes, if you could find a one-to-one mapping between irrational numbers and rational numbers, it would logically mean that the set of real numbers (R) is countable.**

**However, it's crucial to note that this hypothetical scenario is mathematically impossible.** Georg Cantor's diagonal argument rigorously proves that the set of real numbers (R) is **uncountably infinite**. Since R is uncountably infinite and Q is countably infinite, it necessarily follows that the set of irrational numbers (I) must also be uncountably infinite. This means that a one-to-one correspondence between irrational numbers and rational numbers cannot exist.
